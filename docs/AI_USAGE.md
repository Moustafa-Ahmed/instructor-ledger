generally i use ai to help across the life cycle of the project from planning to implementation to testing and documentation.
i use it first to help me understand the requirements and business logic and usually i verify the business logic using normal searching and other ai chats non agents to help me search and summarize results online .
i may ask to implement some of the requirements and bussiness logic that i may not be familiar with to see how it can be implemented and what options do i have to implement .
when i feel i understand the requirements and bussiness logic i will ask it to make a phased plan of implementation for the project and i will verify the plan and make changes to it if needed and then i will ask it to implement the plan in phases and i will verify the implementation of each phase and make changes if needed and then i will ask it to implement the next phase and so on until the project is completed.

as i go on i will use tools to verify the app performance (queries and memory ) and hand test everything and make sure everything works as expected.
